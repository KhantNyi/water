What is WasteBuy?

WasteBuy is a simple concept: people have recyclable waste at home, recycling plants want sorted materials, and someone has to handle the awkward middle step. We make that middle step less chaotic.

Goal

Let users sell recyclables, book a pickup appointment, and estimate pricing online, instead of guessing rates or calling around.

Core features

  • Waste type selection (plastic/paper/metal/glass)
  • Appointment scheduling (date + time window)
  • Estimated payout calculator
  • Final confirmation after real weigh-in

Nice-to-have later

  • User accounts + pickup history
  • Live driver tracking
  • Dynamic pricing by market rate
  • Photo proof / quality grading

How you can extend this

If you’re building this as a term project, the easiest “realistic” step is adding a tiny backend: a REST API endpoint like /appointments and /rates, then store data in a database table. You can keep the UI exactly the same.
